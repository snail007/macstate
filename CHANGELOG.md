# Changelog

## [v1.2.0] - 2026-03-13

### 新增 / Added

- **Finder 右键菜单** — 基于 FinderSync 扩展，右键文件/文件夹/空白处
  - "在此打开终端" — 在选中的目录（或文件所在目录）打开 Terminal.app
  - "复制路径" — 将完整路径复制到剪贴板，多选时逐行复制
  - 通过 DistributedNotificationCenter 实现扩展与主 app 的进程间通信
  - 打开的终端与系统原生行为完全一致（标题、提示符、关闭行为）
- **右键菜单开关** — 设置面板新增"Finder 右键菜单"开关，一键启用/禁用

### 构建 / Build

- build.sh 新增 FinderSync 扩展编译、签名和打包步骤
- 扩展使用沙盒隔离，仅通过 DistributedNotification 与主 app 通信

---

## [v1.1.0] - 2026-03-12

### 新增 / Added

- **进程排行面板** — 点击 CPU / 内存 / 网络指标，弹出 Top 10 进程列表
  - CPU 面板：按 CPU 占用排序，显示进程名、PID、CPU%、命令行
  - 内存面板：按内存占用排序，显示进程名、PID、内存占用、命令行
  - 网络面板：按连接数排序，显示进程名、PID、连接数、LISTEN 端口、命令行
  - 支持点击列头排序（升序/降序切换）
  - 支持点击"查看"按钮弹出完整命令行
  - 面板标题和列头随语言切换实时更新
- **IP 归属地** — 网络进程面板新增"归属地"列
  - 集成 ip2region 离线数据库（~10MB，纯 C 库，0 网络依赖）
  - 中文模式自动翻译国外国名（United States → 美国，约 70 个国家）
  - 中文模式自动翻译常见 ISP 名称
- **连接服务** — 使用 libproc C API 获取进程网络连接信息（TCP/UDP/LISTEN）
- **可点击标签组件** — ClickableLabel 通用组件，支持进程面板交互

### 优化 / Improved

- **CPU 占用大幅降低** — 从 ~87% 降至接近 0%
  - 状态栏合并为单个 NSStatusItem，使用自定义绘制替代多个独立 item
  - 刷新逻辑从 Task.detached 改为专用 DispatchQueue，避免创建过多线程
  - 增加变化检测，数据未变时跳过 UI 更新
- **Tooltip 交互修复** — 温度/风扇/功率等提示框点击外部区域即可关闭
  - 从 `.transient` 改为 `.applicationDefined`，配合全局鼠标事件监听
- **弹框水平滚动** — 进程面板内容超出宽度时显示水平滚动条
- **国际化完善** — 新增 `commandLine` / `viewButton` / `geoLocation` 等本地化字符串

### 修复 / Fixed

- 弹框标题切换语言后不更新
- 弹框内列头标题切换语言后不更新
- 中文模式下 IP 归属地仍显示英文国名
- Tooltip 需要点击两次才能关闭

### 构建 / Build

- build.sh 新增 ip2region C 源码编译和链接步骤
- 新增 ObjC 桥接头文件支持 Swift 调用 C 库

## [v1.0.0] - 2026-03-10

### 新增 / Added

- **CPU 使用率** — 实时百分比显示
- **CPU 温度** — SMC 读取，无需 root 权限
- **内存占用** — 已用/总量百分比
- **风扇转速** — 支持多风扇 RPM 显示
- **网络速度** — 上传/下载速度，自动单位换算（B/KB/MB）
- **充电功率** — SMC PDTR 实时适配器功率显示
  - 点击查看电池详情（充电功率、协商功率、电池功率、电流、电压、电量、循环次数、健康度）
- **独立设置入口** — 状态栏常驻齿轮图标
  - 各监控模块独立开关
  - 刷新间隔配置（1-10 秒）
  - 开机自启设置
  - 中文/英文切换
- **0 外部依赖** — 纯 Swift + IOKit + SMC，无第三方库
- **轻量高效** — 二进制约 780KB，内存占用极低
